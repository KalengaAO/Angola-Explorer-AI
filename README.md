# 🇦🇴 Angola Explorer AI

## 🌍 Overview

Angola Explorer AI is an AI-powered tourism assistant designed to help travelers discover Angola through personalized recommendations.

Instead of forcing users to navigate through multiple websites, categories, and tourism pages, the platform allows travelers to describe what they want in natural language:

> "I want to visit a peaceful desert location in Angola with an English-speaking guide."

The system analyzes the request, retrieves relevant tourism information, and provides recommendations based on registered destinations, attractions, and guides.
